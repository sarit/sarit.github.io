SARIT

SARIT at GitHub

GitHub hosts the application data and text data for SARIT, which allows you to browse and search a collection of texts in Sanskrit and Prakrit.

SARIT has developed a basic set of guidelines describing how to use the TEI Guidelines for encoding the most common features found in printed editions of Indic texts.

SARIT also drafted guidelines that discuss how to use the TEI for encoding advanced features of editions of Indic texts, especially the encoding of textual variations (source here) All of these guidelines are distributed along with the texts and can be found in the schemas/odd subdirectory.
